## 天氣API規格
### 今明36小時API規格
網址: /api/weather
```python
{
"locationName":locationName,
    "Wx":{
      "Wx_firt": Wx_firt,
      "Wx_second": Wx_second,
      "Wx_third": Wx_third,
    },
    "pop":{
      "pop_firt":pop_firt,
      "pop_second":pop_second,
      "pop_third":pop_third,
    },
    "MinT":{
      "MinT_firt":MinT_firt,
      "MinT_second":MinT_second,
      "MinT_third":MinT_third,
    },
    "CI":{
      "CI_firt":CI_firt,
      "CI_second":CI_second,
      "CI_third":CI_third,
    },
    "MaxT":{
      "MaxT_firt":MaxT_firt,
      "MaxT_second":MaxT_second,
      "MaxT_third":MaxT_third
    }
}
```

### 最新天氣狀況API規格
網址: /api/weather_now
```python
{
"locationName":locationName,
    "Wx": Wx,
    "TEMP":TEMP_value,
    "HUMD":HUMD_value,
    "H_24R":H_24R_value,
}
```
